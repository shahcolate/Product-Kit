# Changelog

All notable changes to ProductKit will be documented in this file.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-03-21

### Added
- MVC template with all five components plus operational metadata
- Worked example: AI-powered product recommendations (full MVC)
- Worked example: payment processing migration (stripped-down, no designer)
- Demo test runner with GoodAgent and BadAgent mock implementations
- Contract, behavioral, boundary, and drift test specs
- GitHub Actions workflows for PR tests and weekly drift detection
- Getting started guide (4-week onboarding)
- RACI ownership table
- Solo builder and no-designer guidance
